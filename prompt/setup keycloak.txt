VITE_KEYCLOAK_REALM
VITE_KEYCLOAK_CLIENT_ID


1.ช่วยทำ frontend react นี้ ให้ทำการ login ด้วย keycloak
2.ลง zustand เพื่อให้เก็บ global state แล้วถ้าจำเป็น
3.ให้ทำการ reload token เรื่อยๆด้วย กันเด้งออกเองขณะใช้งาน
4.มี folder ASap.Frontend\src\app\_stores ให้เอาไว้เก็บ store ของ zustand นะ
5.implement การ logout ที่ปุ่ม logout ในรูปด้วย
